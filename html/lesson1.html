Comments in HTML
In most programming languages there is a feature called comments. These do not get compiled by the programming language and do NOT get seen by the user. In HTML, comments are in the format: 

<!-- Hello! -->
Where 'Hello' can be replaced with your comment. An example of a comment is on the left — in both the <head> and <body> tags are examples of comments. Now, if you switch to the Preview tab you can see that the comments aren't rendered. This is because comments are only for developers to make notes on their code — so if it gets complicated, you can easily keep track of what does what. Comments can also be inline. For example: 
<div><!-- Hello! --></div>

Now, you need to learn the basic HTML syntax.

Syntax
In HTML, syntax is pretty easy as long as you can remember the following rules:
  1. Most HTML elements have a start and end tag. E.g. <hello> to open and </hello> to close.
  2. <br> doesn't need an end tag — it's a self-closing element.
  3. Any items between the two tags are 'nested' inside of the element.
